# Запуск проекта

## Установка и запуск Backend
1. Перейдите в папку backend:
cd backend

2. Установите зависимости:
npm install

3. Запустите сервер в режиме разработки:
npm run dev

4. После запуска сервер будет доступен по адресу (если используется порт 5000):
http://localhost:5000


## Установка и запуск Frontend
1. Перейдите в папку frontend:
cd frontend

2. Установите зависимости:
npm install

3. Запустите сервер в режиме разработки:
npm run dev

4. После запуска сервер будет доступен по адресу (если используется порт 5000):
http://localhost:5173
